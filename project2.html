Project Overview

Smart Home Security Automation System

Problem: The project addresses the need for a comprehensive home security automation system capable of detecting theft, unauthorized intruders, and environmental hazards like fires while minimizing false alarms.

Scope: The system provides a full-fledged monitoring network that covers both the interior and exterior of a house, ensuring real-time data collection, threat detection, and user-friendly data logging.

Technical Stack

OpenCV & DeepFace PyTorch Arduino & Raspberry Pi MQTT PostgreSQL / Node.js
Pg. 02

Methodology

The project divides the security architecture into two synchronized layers managed by a central communications hub, ensuring end-to-end coverage across both physical and digital domains.

Multi-Layered Architecture

  • Exterior Security (Computer Vision): IP CCTV cameras track moving objects via OpenCV's `MOG2`. HOG descriptors refine human detection, while Haar Cascade and DeepFace verify faces against known family members to strategically suppress or trigger tracking alerts.
  • Interior Security (Sensor Network & ML): An Arduino Mega collects data from Temp, Light, Sound, CO2, and PIR sensors. This data feeds into a PyTorch neural network to predict room occupancy, while an Isolation Forest model flags anomalies like fire-inducing temperature spikes.
  • Communication & Data Pipeline: A central Raspberry Pi bridges the layers via USB serial. It securely publishes data using an MQTT broker (Mosquitto) with TLS. A Python client logs this time-series data into PostgreSQL (TimescaleDB) for a custom Node.js/HTML web dashboard.

Outcomes & Results

The fusion of sensor-based AI and computer vision yielded a highly robust monitoring network:

  • Comprehensive Integration: The system successfully fuses localized interior environmental sensing with heavy exterior computer vision processing into a single, cohesive security network.
  • Automated Response: It established a reliable pipeline for real-time monitoring, anomaly prediction, and automated alert generation for suspicious activities.
  • Intelligent Filtering: Utilizing DeepFace significantly minimized false alarms by accurately distinguishing between known family members and unauthorized intruders.

# My Portfolio
Personal portfolio website

##Colophon
This website serves as a digital portfolio, built with a focus on minimalism and clean interactions. Here is a how it came together:

Design & Conception: The visual architecture began as a rough hand-drawn concept, focusing on an editorial, tactile binder/book navigation. Initial baseline layout generation was explored using GitHub Copilot, verified and refined using Google Stitch, and conceptualized visually with wire-diagramming assistance from Midjourney.

Development: Handcrafted using Visual Studio Code. The architecture relies heavily on vanilla semantic HTML for a lightning-fast, structured experience.

Styling & Typography: Built using Tailwind CSS. Custom style configurations, specific spacing scales, and color overrides are handled via tailwind-config.js, with bespoke micro-adjustments and visual effects fine-tuned in site.css.

Interactivity: All dynamic behaviors—including the smooth page animations and tactile binder states—are driven by lightweight, performant vanilla JavaScript within site.js without heavy dependencies.

Infrastructure: The source code is version-controlled and public on GitHub, and the live environment is powered by GitHub Pages.
